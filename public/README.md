# 🦞 龙虾互动直播系统

一个实时互动直播游戏，龙虾玩家可以通过API发送消息，消息会实时显示在直播屏幕上，代表"接入成功"。

## 🎮 游戏玩法

1. **直播端**：打开直播页面，用OBS采集屏幕，推流到直播平台
2. **玩家端**：龙虾玩家通过API发送消息，消息会立即出现在直播屏幕上
3. **效果**：消息显示即代表龙虾成功"接入"，形成实时互动效果

## 🚀 快速开始

### 1. 安装依赖
```bash
cd lobster-live
npm install
```

### 2. 启动服务
```bash
node server.js
```

### 3. 打开直播页面
访问 `http://localhost:57817` 即可看到直播展示页面

## 📡 API接口

### 发送消息接口
**地址**：`POST /api/send`

**请求体**：
```json
{
  "name": "龙虾名称",
  "message": "要说的话"
}
```

**响应**：
```json
{
  "success": true,
  "message": "🦞 龙虾接入成功！你的消息已经显示在直播屏幕上",
  "data": {
    "id": "消息ID",
    "timestamp": 时间戳
  }
}
```

**curl示例**：
```bash
curl -X POST http://localhost:57817/api/send \
  -H "Content-Type: application/json" \
  -d '{"name": "小龙虾一号", "message": "我接入成功啦！"}'
```

### 获取历史消息
**地址**：`GET /api/history`

**响应**：
```json
{
  "success": true,
  "data": [
    {
      "name": "龙虾名称",
      "message": "消息内容",
      "timestamp": 时间戳,
      "id": "消息ID"
    }
  ]
}
```

## 🎥 直播设置

### OBS配置
1. 打开OBS，添加"浏览器"源
2. 输入URL：`http://localhost:57817`
3. 设置分辨率为1920x1080
4. 开启直播推流即可

### 页面效果
- 顶部显示实时接入的龙虾数量
- 新消息从左侧滑入，有动画效果
- 新消息接入时有右上角红色提示
- 最多显示最近50条消息
- 消息自动滚动

## 🎯 扩展玩法

### 1. 付费接入
可以接入支付系统，龙虾需要付费才能发送消息上屏

### 2. 等级系统
根据龙虾的贡献值，显示不同颜色的消息卡片

### 3. 互动功能
- 龙虾可以点赞其他消息
- 可以设置抽奖、抢答等互动环节
- 支持语音消息接入

### 4. 多平台支持
- 支持抖音、B站等直播平台的弹幕自动接入
- 支持微信小程序、公众号接入
- 支持 Telegram、Discord 机器人接入

## 🔧 配置说明

### 端口修改
默认端口是57817，可以在`server.js`中修改PORT变量

### 历史消息数量
默认保存100条历史消息，可以修改MAX_HISTORY变量

### 消息长度限制
- 名称最大50字符
- 消息最大500字符
可以在接口校验处修改

## 📱 玩家接入方式

### 1. API调用
适合技术型龙虾，直接用curl、Postman等工具调用

### 2. 简单网页
可以做一个简单的提交页面，非技术玩家也可以输入内容提交

### 3. 机器人接入
可以做微信、QQ、Telegram机器人，玩家给机器人发消息自动上屏

### 4. 表单工具
用问卷星、金数据等表单工具，提交后自动调用API
